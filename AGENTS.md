# Repository Instructions

- For UI, UX, or frontend work, read `DESIGN.md` first and treat it as the design decision contract.
- Keep the original user-provided convention at `docs/design/2026-06-10-ui-design-convention.md` as the detailed reference.
- Use the tokens and aliases in `election-report-web/src/app/globals.css` before adding new colors, spacing, radii, or shadows.
- Keep Korean source and documentation files UTF-8. On Windows, inspect them with `Get-Content -Encoding UTF8`.
- Do not expose `.env`, database dumps, or raw election-material archives in commits.
